# Проверка выбросов
Чтобы проверить аномалии, я использовала метод IQR:

    import numpy as np
    for col in numeric_cols:
        Q1 = raw_data[col].quantile(0.25)
        Q3 = raw_data[col].quantile(0.75)
        IQR = Q3 - Q1
        outliers = raw_data[(raw_data[col] < Q1 - 1.5 * IQR) | (raw_data[col] > Q3 + 1.5 * IQR)]
        print(f"{col}: выбросов — {len(outliers)}")

✅ Результат: выбросов не обнаружено.

# Связь стоимости сырья и роста производства биоэтанола
Построен график рассеяния (`scatterplot`):
- x - Feedstock_Cost,
- y - Bioethanol_Growth.

Цель - выявить, как изменение стоимости сырья отражается на динамике роста производства.
Цветовая схема: зелёные точки с прозрачностью для читаемости при наложении.
<img width="613" height="473" alt="download" src="https://github.com/user-attachments/assets/746762fe-57db-4450-b42e-8b2fd283396b" />

Полное отсутствие зависимости между стоимостью сырья (Feedstock_Cost) и ростом производства биоэтанола (Bioethanol_Growth). Точки образуют плотное, равномерное облако - никакого тренда, наклона, кластеров или паттернов.

Ключевые наблюдения:
1. Рост производства не зависит от цены сырья
* Даже при Feedstock_Cost = 300 рост может быть 1 или 14
* Даже при Feedstock_Cost = 100 - то же самое
2. Страны не выделяются
* США, Бразилия, Китай, Индия, Германия - все в одном облаке
* Нет страновых кластеров → география не влияет
3. Облако - идеальный "шум"
* Каждая комбинация (Cost, Growth) встречается с одинаковой вероятностью
* Это не реальность, а синтетическая генерация
График - визуальное доказательство: стоимость сырья НЕ влияет на рост производства.

# Barplot - сравнение средних значений
Построен столбчатый график (`barplot`), отображающий усреднённые показатели по выбросfv CO₂ по странам. Позволяет визуально сравнить порядок величин различных показателей.
<img width="837" height="590" alt="download" src="https://github.com/user-attachments/assets/363d2f33-2fd6-4a27-9859-607fadd7fba4" />
Ключевые наблюдения:
1. Нет значимых экологических различий
* Все страны работают в одинаковом диапазоне технологий и сырья.
2. Бразилия - чуть "грязнее"
* Возможно, больше кукурузы vs. тростника, или менее эффективные заводы.
* Но разница слишком мала, чтобы делать выводы.
3. Индия - чуть "чище"
* Может быть связано с сырьём (тростник, отходы), но не критично.
4. График - иллюзия различий
* Разница в 0.6 - в пределах шума при равномерном распределении (1–10).
* Не совсем отражает реальность: в жизни Бразилия (тростник) - чище США (кукуруза).
